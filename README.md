# ric-pac-soe

https://naoyasnow.github.io/ric-pac-soe/

「Ric Pac Soe」は「三目並べ（tic-tac-toe）」と「じゃんけん（rock-paper-scissors）」を組み合わせたオリジナルゲーム。〇×の代わりに設置するのは、グー🪨チョキ✂️パー📜。ターンが来たら、新たにマークを設置するか、隣接するマークを攻撃して、自分の色に塗り替えろ。アイコの場合マークは対消滅し、新たなマークを設置するボーナスが得られる。生き残るには、秩序か、多様性か。支配者は、AIか、人類か。

## ゲームモード
  - 「三目並べ（先手一列）」　縦・横・斜め・いずれか1ラインを塗ったプレイヤーの勝ちです
  - 「九目並べ（一色制圧）」　9マスすべてを塗ったプレイヤーの勝ちです

## 対戦モード
  - 「AI対戦」　ルールベースの初級AIとの対戦を楽しめます
  - 「練習/ローカル対戦」　現在は練習用（一人二役）もしくはローカル対戦用です

## 自分の手番の操作
  - 「🪨」「✂️」「📜」ボタンを押した状態で、空いているマスをクリック／タップすると、新しいマークを設置します
  - 「攻撃」ボタンを押した状態で、(1)任意の自分のマスをタップして、(2)隣接する相手のマスをタップすると、「攻撃」ができます
    - ジャンケンで買った場合は、自分の色とマークで、攻撃したマスを上書きします
    - ジャンケンで負けた場合は、自分のマークだけが削除されます
    - ジャンケンであいこになった場合は、自分のマークと相手のマークの両方が削除され、さらに1マス、任意の場所に新しいマークを設置できます
  - 何も操作したくないときは「パス」をしましょう

## 過去のAI
  - 「点式一号」　2手先までの盤面を予想して単純にスコアベースで動作。三目に強い。ハメ技に弱くてかわいい。
  - 「ウイルス一号」　3手先まで読むことに加え、「攻撃性」が高く、ピンチ／均衡の際には「突然変異（多様化）」（混乱や確殺を狙って場に少ないマークを出す）を起こすことがあり、ユーモラスに転覆をたくらむ。九目に強い。かわいい。
